A session can have a role "ContPrev" that indicates it is a continuation of the chronologically previous session for the same game, just held at a different time.

Same for "ContNext" that indicates it is a continuation of the next session.

Generally these will be the only roles for these continuation sessions.  Other roles should be ignored.

There's also "FinPrev" indicating that this game finishes the previous game.  It has the same semantics as ContPrev, it's just clearer that this was the final session.
